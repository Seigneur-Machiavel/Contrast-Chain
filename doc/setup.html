Contrast-Chain Setup Guide

1. Setting Up on Linux

1.1. Install Git

Git is essential for cloning the repository. Install it using your package manager.

sudo apt update
sudo apt install git -y

1.2. Install NVM (Node Version Manager)

NVM allows you to manage multiple Node.js versions.

curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.1/install.sh | bash

After installation, either restart your terminal or run the following commands to load NVM:

export NVM_DIR="$([ -z "${XDG_CONFIG_HOME-}" ] && printf %s "${HOME}/.nvm" || printf %s "${XDG_CONFIG_HOME}/nvm")"
[ -s "$NVM_DIR/nvm.sh" ] && \. "$NVM_DIR/nvm.sh"

1.3. Install the Latest Node.js Version

nvm install node

1.4. Clone the Repository

git clone https://github.com/Seigneur-Machiavel/Contrast-Chain

1.5. Navigate to the Project Folder and Install Dependencies

cd Contrast-Chain
npm install

1.6. Create Blockchain Folder (Optional)

The following step creates a blockchain directory structure. It might be optional depending on your setup.

mkdir -p nodes-data/W8tPnfZiU2GLSLBQgRS6/blockchain

1.7. Testing the Setup

Choose one of the following options to test the setup:

  • node run/pureObserver.mjs
  • node run/dashboard.mjs
  • node run/runMultiNodeWithDashboard.mjs
  • node test_local

1.8. Running the Application

Run the application in the background using nohup:

  • nohup node run/pureObserver.mjs > output.log 2>&1 &
  • nohup node run/dashboard.mjs > output.log 2>&1 &
  • nohup node run/runMultiNodeWithDashboard.mjs > output.log 2>&1 &

1.9. Deleting the Contrast-Chain Repository Folder

If you wish to remove the repository folder:

rm -rf Contrast-Chain

2. Setting Up on Windows

2.1. Install Git

Download and install Git from the official website:

Download Git for Windows

Follow the installation prompts to complete the setup.

2.2. Install Node.js

Download and install Node.js from the official website:

Download Node.js

The installer includes npm, which is required for managing project dependencies.

2.3. Install PM2

PM2 is a process manager for Node.js applications. Install it globally using npm:

npm install pm2 -g

2.4. Clone the Repository

git clone https://github.com/Seigneur-Machiavel/Contrast-Chain

2.5. Navigate to the Project Folder and Install Dependencies

cd Contrast-Chain
npm install

2.6. Create Blockchain Folder (Optional)

The following step creates a blockchain directory structure. It might be optional depending on your setup.

mkdir nodes-data\W8tPnfZiU2GLSLBQgRS6\blockchain

2.7. Testing the Setup

Choose one of the following options to test the setup:

  • node run/pureObserver.mjs
  • node run/dashboard.mjs
  • node run/runMultiNodeWithDashboard.mjs
  • node test_local

2.8. Running the Application with PM2

Use PM2 to manage and keep the application running in the background:

  • pm2 start run/pureObserver.mjs --name pureObserver
  • pm2 start run/dashboard.mjs --name dashboard
  • pm2 start run/runMultiNodeWithDashboard.mjs --name multiNodeDashboard

View logs with:

pm2 logs

2.9. Deleting the Contrast-Chain Repository Folder

If you wish to remove the repository folder:

rmdir /s /q Contrast-Chain

3. Additional Notes

  • Environment Variables: Ensure that any required environment variables are set appropriately.
  • Firewall Settings: Configure your firewall to allow necessary ports if you're running a server.
  • Documentation: Refer to the Contrast-Chain GitHub repository for more detailed information and updates.

4. Troubleshooting

  • Permission Issues: If you encounter permission errors, try running commands with elevated privileges.
  • Dependency Errors: Ensure that all dependencies are correctly installed. You can try deleting the node_modules folder and running npm install again.
  • Port Conflicts: If the application fails to start due to port conflicts, identify and free up the required ports or configure the application to use different ones.
  • Log Files: Check log files for detailed error messages to assist in debugging.
